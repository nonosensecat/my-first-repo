# 🧪 實驗報告：水中溶氧檢測－疊氮化物法 (NIEA W421.56C)

> **實驗日期：** 2025 年 3 月 13 日
> **方法依據：** 環境部環境檢驗所 NIEA W421.56C

---

## 壹、方法概要
本方法為**碘定量法**之修正方式。碘定量法係最精確及可靠的溶氧滴定方法：
1. **反應原理**：將樣品盛裝於密封玻璃瓶中，加入硫酸亞錳及含碘離子的鹼性溶液，水中溶氧會將氫氧化亞錳沉澱氧化成高價氫氧化物。
2. **酸化與滴定**：當水樣酸化後，高價氫氧化物氧化碘離子生成與溶氧相同當量之 **碘 (I2)**，再以**硫代硫酸鈉 (Na2S2O3)** 溶液滴定。
3. **干擾消除**：亞硝酸鹽會造成循環性干擾（藍色褪色後再生成），加入**疊氮化物**可有效去除此影響。

## 參、試劑
1. **試劑水**
2. **濃硫酸** (98%)
3. **硫酸亞錳溶液**
4. **鹼性碘化物－疊氮化物試劑**
5. **澱粉指示劑** (作終點偵測用)
6. **碘酸氫鉀標準溶液** (0.00208 M)
7. **硫代硫酸鈉滴定溶液** (0.025 M)
8. **氟化鉀溶液** (當三價鐵離子 > 5 mg/L 時使用)

---

## 肆、實驗步驟



1. **加藥**：於滿水樣之 BOD 瓶加入 **1 mL** 硫酸亞錳與 **1 mL** 鹼性碘化物－疊氮化物試劑。
2. **混合**：加蓋（勿留氣泡）上下倒置數次，待棕色沉澱下沉至半瓶。
3. **酸化**：加入 **1 mL** 濃硫酸，倒置混勻直到沉澱完全溶解。
4. **滴定**：取 **201 mL** 水樣（相當於 200 mL 原水樣），以 **0.025 M** 硫代硫酸鈉滴定至**淡黃色**。
5. **終點**：加數滴澱粉指示劑，滴定至**藍色消失**。

## 伍、結果與處理

### 一、 計算公式

$$溶氧量 (mg\,O_2/L) = \frac{A \times N \times 8000}{V_1} \times \frac{V}{V - V_2}$$

* **A**：水樣消耗之硫代硫酸鈉體積 (mL)
* **N**：硫代硫酸鈉當量濃度 (0.025 N)
* **V1**：滴定用的水樣準確體積 (201 mL)
* **V**：BOD 瓶總體積 (300 mL)
* **V2**：加入試劑之總體積 (2 mL)

### 二、 數據紀錄與 BOD5 計算

| 觀測階段 | 編號 | 滴定起點 (mL) | 滴定終點 (mL) | 消耗體積 A (mL) | 溶氧量 (mg/L) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **D1** (3/7) | 368 | 0.00 | 5.6 | 5.6 | **5.638** |
| **D1** (3/7) | 115 | 0.00 | 7.1 | 7.1 | **7.148** |
| **D2** (3/12)| 379 | 2.00 | 8.2 | 6.2 | **6.242** |
| **D2** (3/12)| 162 | 9.00 | 15.4 | 6.4 | **6.443** |

* **平均 D1 (初日)**：(5.638 + 7.148) / 2 = **6.393 mg/L**
* **平均 D2 (五日)**：(6.242 + 6.443) / 2 = **6.342 mg/L**
* **BOD5 計算** (P = 0.001):
  (6.393 - 6.342) / 0.001 = **51 mg/L**

## 柒、備註
1. **取樣校正**：取 **201 mL** 是為了補償加入 **2 mL** 試劑後排出的水量。
   計算式：$X = \frac{200 \times 300}{300 - 2} = 201\,mL$
2. **終點回色**：若溶液在滴定後又變回藍色，通常是亞硝酸鹽或鐵離子造成的催化效應。
